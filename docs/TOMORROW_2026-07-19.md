## 🎯 Manager Investigation SQL (Reference)

### Suspicious cancellation patterns
SELECT cashierId, COUNT(*) as cancels, SUM(total) as lost
FROM held_transactions
WHERE status = 'CANCELLED' AND heldAt >= date('now', '-7 days')
GROUP BY cashierId
HAVING cancels > 5;

### Cashier performance
SELECT cashierName,
  SUM(CASE WHEN status='COMPLETED' THEN 1 ELSE 0 END) as completed,
  SUM(CASE WHEN status='CANCELLED' THEN 1 ELSE 0 END) as cancelled,
  SUM(CASE WHEN status='EXPIRED' THEN 1 ELSE 0 END) as expired
FROM held_transactions
GROUP BY cashierName;
